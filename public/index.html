Test API

Test API is a free online service that helps you to test your Http calls. Feel free to use it for unit testing of front-end applications, making tutorials, examples, studying or whatever you want.

Routes

/echo

Echo API allows you to send request and to get in response what you sent. It is useful for simple unit tests. Allows GET, POST, PUT, PATCH and DELETE methods.

/rest

REST API allows you to send full spector of requests using REST standards. Use /rest/users route to work with users mock up.

Use GET method with no options to get all of 1000 records or use page_index and page_size query parameters to achieve pagination output.

Example with pagination: http://test-api.tech/rest/users?page_index=12&page_size=30

SSL Support

API is accessible with and without TLS. Feel free to use both http and https according to your needs.
